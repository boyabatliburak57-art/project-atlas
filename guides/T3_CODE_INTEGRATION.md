# T3 Code Entegrasyon Rehberi

## 1. Repository'yi aç

T3 Code içinde `project-atlas` klasörünü workspace olarak aç.

## 2. İlk okuma

`T3_CODE_START_HERE.md` içindeki ilk oturum mesajını gönder.

T3 Code özetinde şunlar bulunmalıdır:

- ilk sürümün BIST odaklı olması
- modüler monolith yaklaşımı
- veri sağlayıcı adapter'ları
- tarama kuralı AST modeli
- kodlamaya henüz başlamaması

## 3. İlk görevi çalıştır

```text
tasks/TASK-001-Repository-Validation.md dosyasını uygula.
```

## 4. Değişiklik kontrolü

T3 Code'dan şunu iste:

```text
Değişen dosyaları tam yollarıyla listele. Çalıştırılan komutları ve test sonuçlarını ver. Dokümanlara aykırı geçici çözüm eklenip eklenmediğini belirt.
```

Sonra terminalde:

```bash
git status
git diff
```

## 5. Yeni oturumlarda

```text
Önce ATLAS_INDEX.md, SYSTEM_PROMPT.md ve üzerinde çalışacağımız görev kartını tekrar oku. Önceki oturum varsayımlarına güvenme.
```
